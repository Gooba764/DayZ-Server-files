REQUIRED MODS FOR THE FILES TO WORK (all of which can be found rather easily on the DayZ workshop)
Field Operator Gear By AJ
AJ's Weapons By AJ as the name suggests
DayZ Horse (Requires Survivor animations however this is optional, i havent had any problems not loading this mod) By HunterZ
sGunplay, sVisuals, sFramework By Simonvic all are interchangable and not necessary for the function of these files however they make the game much better.
Inedia Infected AI and Inedia Stamina By johnkind also optional.
Other than these few mods these map files are incredibly versatile, all i ask is that you have fun and do a better job of adding items than i did!
